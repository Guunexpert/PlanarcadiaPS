# How to Run (Windows)
1. Install [zig 0.14.1](https://ziglang.org/download/0.14.1/zig-x86_64-windows-0.14.1.zip)
2. Build
- Run the server ```zig build run-program```
- Build release ```zig build -Doptimize=ReleaseSafe```
